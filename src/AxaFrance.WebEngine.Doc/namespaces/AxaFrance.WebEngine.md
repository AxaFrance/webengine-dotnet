The AxaFrance.WebEngine namespace contains classes and interfaces of basic structure of Test. The classes used in this namespace are platform and underlying tool independent.

To test Web Based applications, You can use the functionalities provided in <xref:AxaFrance.WebEngine.Web> namespace. This libaray uses Selenium as underlying tool to preform test on Desktop and Mobile Browsers (using Selenium Grid or local Appium Server)

To test Native/Hybird Mobile Applications running on `Android` or `iOS` devices, You can use functionalities provided in <xref:AxaFrance.WebEngine.MobileApp>. This library uses Appium as underlying tool.
